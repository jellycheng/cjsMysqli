# cjsMysqli


## php mysqli
```
http://php.net/manual/zh/mysqli.summary.php
mysqli提供面向过程和面向对象的2种方式接口
MySQLi面向过程的方法以mysqli_ 和 mysqli_stmt_开头的函数
MySQLi面向对象的三个常用类：
	1. mysqli 进行db连接，关闭，选库，执行sql和预执行sql操作
	2. mysqli_result 结果集处理类（对select，show，desc等sql）
	3. mysqli_stmt 执行预处理sql，返回的对象，该类是对预处理结果集的操作



```
